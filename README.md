# SampleGraphQL
The very basics of building an Android client for the GraphQL API with Apollo.

## Requirements
* Android Studio 3.5 Canary 6
* Kotlin 1.3.30-eap-11
* Android Gradle Plugin 3.5.0-alpha06
* Gradle wrapper 5.2.1
* AAPT 2
